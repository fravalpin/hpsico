# Landing Page Psicóloga Ana Herrera

Esta es una **landing page one-page** desarrollada para la psicóloga sanitaria **Ana Herrera**. El objetivo de la web es presentar sus servicios, enfoque terapéutico y facilitar el contacto con pacientes potenciales de una manera cercana y profesional.

## 🚀 Tecnologías Utilizadas

*   **React** (vía Vite) - Framework de JavaScript para la interfaz de usuario.
*   **Bootstrap 5** & **React-Bootstrap** - Framework CSS y componentes para diseño responsive y consistente.
*   **CSS3** - Variables CSS para tematización personalizada.

## 📄 Estructura y Funcionalidades

La web sigue una arquitectura **One-Page**, donde todo el contenido reside en una única página dividida en secciones lógicas. La navegación se realiza mediante un menú sticky que desplaza suavemente la vista (smooth scroll) hacia las anclas correspondientes.

### Secciones
1.  **Inicio (Hero):** Bienvenida, introducción al enfoque integrador y llamada a la acción principal.
2.  **Sobre Mí:** Filosofía de trabajo, trayectoria y enfoque personal.
3.  **Servicios / Terapias:** Lista detallada de motivos de consulta y modalidades (online, presencial, domicilio).
4.  **FAQ:** Respuestas a preguntas frecuentes sobre el proceso terapéutico.
5.  **Contacto:** Información de contacto, formulario y acceso directo a WhatsApp.

## 🎨 Diseño y Estilo

*   **Enfoque Mobile-First:** Diseñada prioritariamente para ofrecer una experiencia perfecta en dispositivos móviles, escalando elegantemente a pantallas de escritorio.
*   **Paleta de Colores:** Colores suaves y profesionales definidos por la marca:
    *   Fondo: `#f2f2f2`
    *   Primario (Botones/Títulos): `#516b8c`
    *   Acento: `#8fa4bf`
    *   Texto: `#0d0d0d`

## 🛠️ Instalación y Ejecución

Para ejecutar este proyecto en local:

1.  **Clonar el repositorio:**
    ```bash
    git clone <URL_DEL_REPOSITORIO>
    cd anapsico
    ```

2.  **Instalar dependencias:**
    ```bash
    npm install
    ```

3.  **Ejecutar servidor de desarrollo:**
    ```bash
    npm run dev
    ```
    La aplicación estará disponible en `http://localhost:5173`.
